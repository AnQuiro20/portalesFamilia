Portales de Belén Familiares

La familia Villalobos Rojas, con mucha alegría, desea compartir estos nacimientos elaborados en el seno familiar como un signo de fe, esperanza y del rescate de tan valiosa tradición. Asimismo, buscamos motivar a las futuras generaciones a continuar con este hermoso símbolo. Esta es una colección especial de portales navideños creados por familias para celebrar la magia de la Navidad.

Cómo agregar más imágenes a cada familia

Para agregar más fotos o videos a cada portal familiar:

  1. Añade la propiedad "gallery" a cada objeto en el array portalsData
  2. La propiedad "gallery" debe ser un array con las rutas de tus imágenes/videos
  3. Para videos, usa objetos con propiedades "type": "video" y "src": "ruta/al/video.mp4"
// Ejemplo:
{
  id: 1,
  family: "Familia Porras Moreno",
  location: "Tu ubicación",
  image: "Portales/Porras Moreno.jpeg",
  gallery: [
    "Portales/Porras Moreno.jpeg",
    "Portales/Porras Moreno-2.jpg",
    {type: "video", src: "videos/familia-porras.mp4"}
  ],
  year: 2025
}
